Jak to działa

1. Firefox Plugin

Plugin do Firefoxa automatycznie wykrywa fragmenty kodu React na stronach internetowych i dodaje przycisk "Renderuj", który otwiera popup z wyrenderowanym komponentem.

  • Automatyczna detekcja kodu React
  • Renderowanie kodu w popupie
  • Kopiowanie wyrenderowanego HTML do schowka
  • Obsługa błędów renderowania

Render Server
// Serwer Express do renderowania komponentów React
const express = require('express');
const React = require('react');
const ReactDOMServer = require('react-dom/server');

app.post('/render', (req, res) => {
  try {
    const { code } = req.body;
    // Transformacja i renderowanie komponentu React
    const component = eval(code);
    const html = ReactDOMServer.renderToString(
      React.createElement(component)
    );
    res.json({ html });
  } catch (error) {
    res.status(400).json({ error: error.message });
  }
});

2. Render Server

Lokalny serwer Express, który przetwarza i renderuje kod JavaScript z React. Serwer przyjmuje kod komponentu React, kompiluje go i zwraca wyrenderowany HTML.

  • Transformacja kodu przez Babel
  • Renderowanie komponentów React
  • Obsługa hooków React
  • Wsparcie dla Tailwind CSS
  • Komunikacja przez API

Instalacja i użycie

Firefox Plugin

Instalacja tymczasowa:

  1. Sklonuj repozytorium git clone https://github.com/comarkup/firefox
  2. W Firefox przejdź do about:debugging
  3. Kliknij "This Firefox"
  4. Kliknij "Load Temporary Add-on"
  5. Wybierz plik manifest.json z katalogu

Instalacja z marketplace:

  1. Odwiedź stronę dodatku w Firefox Add-ons
  2. Kliknij "Add to Firefox"
  3. Potwierdź instalację

Render Server

Instalacja i uruchomienie:

  1. Sklonuj repozytorium git clone https://github.com/comarkup/render-react
  2. Nadaj uprawnienia do wykonania: chmod +x setup.sh
  3. Uruchom skrypt instalacyjny: ./setup.sh
  4. Uruchom serwer: npm start

Testowanie API:

Użyj curl do przetestowania API:

curl -X POST \
  http://localhost:3001/render \
  -H "Content-Type: application/json" \
  -d '{"code": "function ExampleComponent() { return React.createElement('\''div'\'', null, '\''Hello'\'') }"}'
